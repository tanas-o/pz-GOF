# GOF Patterns in TypeScript

## Реалізовані патерни

### Creational
1. **Singleton (Cloud Telemetry)**: Гарантує єдиний екземпляр для збору метрик у розподіленій системі.
2. **Builder (Financial Report)**: Дозволяє створювати складні фінансові звіти з багатьма секціями покроково.

### Structural
1. **Adapter (Legacy Bank)**: Адаптує старі XML-дані банківської системи до сучасного JSON-інтерфейсу.
2. **Facade (Smart Home)**: Надає простий інтерфейс ("Sleep Mode") для керування групою складних підсистем.

### Behavioral
1. **Observer (Crypto Alerts)**: Реалізує подієву модель сповіщень про зміну ціни активів без постійних запитів до сервера.
2. **Strategy (Logistics)**: Дозволяє динамічно змінювати алгоритм розрахунку вартості доставки залежно від обраного перевізника.
